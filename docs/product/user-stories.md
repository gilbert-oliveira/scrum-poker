# Histórias de Usuário - Scrum Poker

## EPIC-01 Infraestrutura

- **US-01.1**: Como desenvolvedor, quero configurar o projeto Next.js com TailwindCSS para acelerar o desenvolvimento.
- **US-01.2**: Como desenvolvedor, quero configurar o projeto no Firebase (Firestore) para ter persistência de dados.

## EPIC-02 Scrum Poker Core

### Criação de Sala

- **US-02.1**: Como facilitador, quero criar uma sala com um nome personalizado para identificar a sessão.
- **US-02.2**: Como facilitador, quero escolher o tipo de escala (Fibonacci, T-Shirt) para adequar ao time.
- **US-02.3**: Como facilitador, quero um link compartilhável da sala para enviar aos participantes.

### Entrada na Sala

- **US-03.1**: Como participante, quero entrar na sala informando apenas meu nome (sem login) para agilidade (Modo Convidado).
- **US-03.2**: Como participante, quero ter um avatar gerado automaticamente para me identificar visualmente.

### Votação

- **US-04.1**: Como participante, quero selecionar uma carta da escala escolhida para estimar a tarefa.
- **US-04.2**: Como participante, quero ver quem já votou (sem ver o voto) para saber o progresso da rodada.
- **US-04.3**: Como participante, quero poder alterar meu voto antes do reveal caso mude de ideia.

### Resultado / Reveal

- **US-05.1**: Como facilitador, quero revelar os votos de todos simultaneamente para evitar viés.
- **US-05.2**: Como facilitador, quero ver a média e estatísticas dos votos para auxiliar na decisão.
- **US-05.3**: Como facilitador, quero iniciar uma nova rodada (resetar votos) para a próxima tarefa.

## EPIC-06 Times (SaaS)

- **US-06.1**: Como usuário logado, quero criar um time para agrupar meus membros.
- **US-06.2**: Como admin de time, quero convidar membros por email para dar acesso às salas do time.

## EPIC-07 Pagamentos (SaaS)

- **US-07.1**: Como usuário, quero assinar um plano Pro para desbloquear histórico ilimitado.
- **US-07.2**: Como usuário, quero poder cancelar minha assinatura a qualquer momento.
